# Task Manager (Python CLI App)

## Description
A simple command-line Task Manager built using Python.
This app allows users to add, view, edit, and delete tasks.
Tasks are stored in a JSON file for persistence.

## Features
- Add tasks
- View tasks
- Edit tasks
- Delete tasks
- JSON storage

## Technologies Used
- Python
- JSON
- File handling

## How to Run
1. Make sure Python is installed
2. Run:
   python task_manager.py
